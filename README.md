# 💱 Currency Converter

A simple and responsive currency converter web app that fetches real-time exchange rates using an API.

## 🚀 Live Demo

https://currency-converter-olive-omega.vercel.app/

## 📌 Features

* Convert between different currencies
* Real-time exchange rates using API
* Easy-to-use interface
* Country flags for better UX

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Exchange Rate API

## 📂 Project Structure

* index.html
* style.css
* app.js
* codes.js

## ⚡ How to Run Locally

1. Clone the repository
2. Open index.html in browser
